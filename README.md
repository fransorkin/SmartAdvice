# Smart Advice — Landing

Landing page de **Smart Advice**, el copiloto empresarial con IA para PYMEs españolas
(asesoría legal, industrial, ciberseguridad y seguros, con respaldo de profesionales reales).

## Contenido

- `index.html` — Landing completa en **React** (vía CDN: React 18 + Babel standalone).
  Tipografía Spectral + Libre Franklin, estilos propios, botón flotante de WhatsApp
  y todos los CTA enlazados a la app.

## Cómo verla

Abre `index.html` en cualquier navegador (doble clic). No requiere instalación ni build.

## Enlaces de los botones

- Registro / prueba gratis / planes → `https://app.smrtadvice.es/auth/register`
- Iniciar sesión → `https://app.smrtadvice.es/auth/login`
- Contacto → `contacto@smartadvice.es`
- WhatsApp → `https://wa.me/34611403013`
- Navegación interna → anclas `#features`, `#como-funciona`, `#precios`
